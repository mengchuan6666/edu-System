# 基于springboot+layui的在线教育平台系统

#### 介绍
springboot+layui在线教育平台系统

## 1. 技术介绍

核心技术：SpringBoot+mybatis;

前端：layui；

开发工具：idea；

数据库：mysql5.7；

模版引擎：thymeleaf；

安全框架：SpringSecurity；

日志框架：logback；

数据库连接池：druid；

在线编辑器：ckeditor；

图片轮播组件：jQuerySwipeslider；
